## Lectura

La suite no se presenta como salvación.

Se presenta como una manera de mirar.

Una herramienta para cada clase de residuo.  
Un nombre para cada habitación.  
Una marca para cada cosa que el sistema intenta dejar atrás.

OXLGR no promete pureza.

Promete borde.
